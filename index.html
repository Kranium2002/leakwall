LeakWall

AI agent security platform. Protect your coding agents from secret leaks, tool poisoning, and behavioral anomalies.

Secret Redaction

MITM proxy intercepts LLM API calls and strips leaked credentials before they leave your machine.

MCP Auditing

Scans MCP server configs for tool poisoning, known CVEs, and rug pulls with hash-pin detection.

Behavioral Analysis

Profiles outbound traffic for volume spikes, new destinations, and high-entropy secret-shaped data.

Filesystem Monitoring

Watch daemon detects changes to .env files, MCP configs, skill files, and credentials in real-time.

Install

Shell (Linux / macOS)

curl -sSL https://kranium2002.github.io/leakwall/install.sh | sh

PowerShell (Windows)

irm https://kranium2002.github.io/leakwall/install.ps1 | iex

APT (Debian / Ubuntu)

echo "deb [trusted=yes] https://kranium2002.github.io/leakwall/deb stable main" \
  | sudo tee /etc/apt/sources.list.d/leakwall.list
sudo apt update
sudo apt install leakwall

Homebrew (macOS / Linux)

brew tap Kranium2002/leakwall https://github.com/Kranium2002/leakwall.git
brew install leakwall

Cargo

cargo install leakwall

From Source

git clone https://github.com/Kranium2002/leakwall.git
cd leakwall
cargo install --path crates/leakwall-cli

Quick Start

# Protect any AI agent session (redact mode is default)
leakwall run -- claude

# Block mode — drop requests containing secrets
leakwall run -m block -- cursor

# With TUI dashboard
leakwall run -T -- claude

# Audit MCP servers and credentials
leakwall scan

# Monitor filesystem for changes
leakwall watch --daemon

# Generate HTML report
leakwall report --html --open

How It Works

          Your Agent (Claude, Cursor, Copilot, etc.)
    |
    | HTTPS requests via HTTPS_PROXY
    v
+-------------------------------+
|         LeakWall Proxy        |
|    127.0.0.1:<port>          |
|                               |
|  1. TLS intercept (MITM)      |
|  2. Scan body for secrets     |
|  3. Redact / Block / Warn     |
|  4. Forward to real API       |
+-------------------------------+
    |
    v
LLM API (Anthropic, OpenAI, Groq, Mistral, ...)

Two-Layer Secret Detection

Layer Method What It Catches
Layer 1 Aho-Corasick fingerprinting Known secrets from .env, cloud creds, env vars — in raw, base64, URL-encoded, and JSON-escaped forms
Layer 2 Regex pattern matching 17 format-based patterns: AWS keys, GitHub PATs, Stripe keys, JWTs, Google API keys, and more

Scan Modes

ModeBehavior
redact (default) Replace secrets with [LEAKWALL:pattern:REDACTED], forward sanitized request
warn Detect and log secrets, forward request unchanged
block Drop the entire request, return 403

Intercepted APIs

api.anthropic.com api.openai.com api.groq.com api.mistral.ai api.together.xyz api.fireworks.ai generativelanguage.googleapis.com api.cohere.com api.deepseek.com

All other HTTPS traffic passes through without inspection.
